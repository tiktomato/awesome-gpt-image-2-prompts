## GPT Image 2 について

OpenAI のフラッグシップ画像生成モデル（2026 年 4 月リリース）。前世代の画像モデルと比べ、GPT Image 2 は次の点で際立っています：

- **ピクセル精度のタイポグラフィ** — 画像内テキスト、ブランドロゴ、複雑なレイアウトを鮮明にレンダリング
- **画像間の一貫性** — 編集をまたいで顔、商品、スタイルを保持
- **複数画像入力** — 参照画像をインテリジェントに組み合わせ
- **フォトリアルな出力** — 広告、ポスター、雑誌表紙にそのまま使える商業品質
