Below the headline, include 4 clean icon-based feature badges in a row with text such as "NO ADDED SUGAR," "100% NATURAL," "NO PRESERVATIVES," and "NOT FROM CONCENTRATE." At the bottom right, display three smaller bottle variants in different flavors neatly arranged. Use soft natural lighting mixed with commercial studio polish, realistic shadows, shallow depth of field, glossy floor reflections, and a premium energetic eCommerce campaign aesthetic inspired by modern Instagram and billboard ads.
