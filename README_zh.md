## 关于 GPT Image 2

OpenAI 旗舰图片生成模型（2026 年 4 月发布）。相比上一代图片模型，GPT Image 2 的突出能力：

- **像素级文字渲染** — 海报字面文案、品牌 Logo、复杂版式都能清晰呈现
- **跨图一致性** — 多次编辑保持人脸、产品、风格一致
- **多图输入** — 智能融合多张参考图
- **写实级输出** — 商业广告级质量，可直接用于海报、杂志封面
